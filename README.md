# Certificate Generator

A desktop application that can generate certificates in bulk for different individuals. Provided a template and names, it allows you to generate personalized certificates easily. The application provides a GUI for selecting the area where the name should be written, along with options for selecting font size and color for the text.

## Features

- Bulk certificate generation
- User-friendly GUI
- Customizable text area
- Font size and color selection

## Usage

1. Open the application.
2. Select the certificate template.
3. Define the area where the name should be written.
4. Choose the font size and color for the text.
5. Provide the list of names.
6. Generate the certificates in bulk.
